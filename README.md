# SpeedRunDeck
Recently, Iâ€™ve gotten into Minecraft speedrunning, but I quickly noticed that you need to press a lot of difficult keybinds to make your runs efficient and fast. For example, F3 + F or F3 + Shift + F is used to quickly increase or decrease the render distance in Minecraft, which is useful for pearl hanging, calculating the distance to the blaze spawner, and many other speedrunning strategies. While doing this, you also need to keep moving and perform other tasks, which makes pressing these keybinds very difficult.

This is why I am creating my Speedrun Deck, which you can customize to perform any keyboard shortcut with a single button for Minecraft. This is my first time working with electronics, so I hope this will be a good first project.
